# MELON APP

An AI-powered learning application for children.

## Features
- AI-generated lessons and quizzes
- Personalized learning paths
- Parent monitoring dashboard
- Gamification system

## Tech Stack
- Frontend: React / Flutter
- Backend: Node.js / Spring Boot
- AI: OpenAI API / LLM
